Agent-native by design

Built for autonomous workflows: fetch context, score risk, and ship structured outputs to agents.

Morning Risk Scan

$ pftui brief --agent --json

Generate machine-readable portfolio state for your daily decision loop.

Macro + Sentiment Pulse

$ pftui macro && pftui predictions --limit 10

Combine rates, volatility, dollar strength, and crowd probabilities in seconds.

Export for Automation

$ pftui export json && pftui summary --period 1m

Push clean data to scripts, cron jobs, or LLM pipelines without UI overhead.
