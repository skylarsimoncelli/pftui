See it in action

Fully agent-native, fully agent-driven

pftui is designed to be operated by your AI agent. Every feature has a CLI counterpart with --json output. The agent guide (AGENTS.md) covers setup, daily operation, and long-term system design — so any agent can run pftui as thoroughly as a human analyst.

One-Command Intelligence

$ pftui refresh && pftui brief --agent --json

Your agent fetches 10+ data sources and gets a complete portfolio snapshot — positions, macro, news, sentiment, predictions, movers — in one JSON blob. No web searching required.

Agent-Guided Setup

"Read AGENTS.md and set up pftui for me"

Tell Claude Code, Codex, or OpenClaw to read the agent guide. It will ask about your holdings, risk tolerance, and market views — then populate everything automatically.

Automated Daily Briefs

$ pftui eod && pftui movers --json

Set up cron jobs for morning briefs and market close summaries. Your agent monitors, analyses, and delivers — you read on your phone or terminal.
